Overview

This project is a simple, secure digital journal platform designed for easy writing and organization. It offers a clean, distraction-free interface with features like mood tracking, search, and customizable entries to encourage regular journaling and self-reflection.

Problem

The existing Penzu journal app faced several user experience challenges. Its interface lacked visual harmony and failed to deliver a calming environment suitable for personal journaling. The onboarding process appeared outdated, cluttered, and disconnected from the appâ€™s core purpose, making it less engaging for new users. As a result, users reported a low emotional connection with the app, describing it as uninviting. Additionally, poor visibility of key features contributed to low engagement levels and weak user retention rates.

Solution

1. Redesigned the onboarding screen with a clean, calming visual layout using a soft blue color palette.

2. Simplified the sign-in/sign-up flow with improved button hierarchy and clear CTAs.

3. Created a consistent design system to enhance usability and visual identity across the app.

4. Focused on emotional design to build a stronger personal connection and increase user engagement.

User Testing (A/B Testing for Redesign)

Conducted A/B testing on two versions of the Record Page and Sign In/Sign Up screens.

Tested visual changes like font type (Playfair), color palette (monochrome blues), and added features (text alignment, stickers, background themes).

Shared mockups with a small group of users and collected feedback via Google Forms and user interviews.

Measured engagement (scroll rate, time spent, feature usage) to compare responses to original vs. redesigned versions.

Key Findings from A/B Testing:

  • Users preferred the monochromatic blue theme for its calming and focused mood.
  • The new font and alignment options improved readability and writing comfort.
  • Majority of users found the sticker and background features fun and expressive.
  • The updated Sign In/Sign Up flow led to a smoother onboarding experience.
  • Usability score improved with clearer navigation and better layout structure.

Key Takeaways

Personalization Increases User Engagement: Customizable features like font styles, backgrounds, and stickers helped users feel more connected to their journal entries.

Consistent Design System Enhances Trust: A unified visual system made the app feel more polished and reliable across all pages.

User-Centered Design Outperforms Assumptions: Designs that were guided by actual user feedback worked better than those based solely on personal preference.

CA/B Testing Validates Design Decisions: Side-by-side testing of design variations helped confirm which version users preferred before making final changes.

Small UI Tweaks Have Big Impact: Adjustments like improved button placements and spacing improved overall usability and aesthetics.
